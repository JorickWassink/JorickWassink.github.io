Monkey Survivors

Description

Monkey Survivors is a group project based a lot off of the game vampire survivors. It is a mobile game where you fight through hordes of enemies trying your best to survive for as long as possible.

To set it apart from other games like it we added a crafting system where enemies drop materials that can be used to craft other weapons. It also features simple map generation by randomly generation the background of the game and some obstacles.
