Limbound

Description

Limbound is a very simple 2d platformer made in unity where the player needs to use light to see where they have to go. Limbound was made as a school assignment where the goal was to create a simple 2d platformer with a unique point to it.

I chose to use the lighing feature because i wanted to try using the URP lighting engine instead of the base one that unity has. The lighting feature also makes for a fun and frustrating game where you need to keep finding out where to go.

 

Under here are some systems in the game

Lighting

In limbound you need to find where to go using light that you can move with your mouse

Spiked balls

In limbound there are spiked balls that when touched put the player back to the last piece of ground they touched.

Slopes

In limbound there are slopes that stop the players movement and slowly slide them down.
